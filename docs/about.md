# About {{ site.serverName }}

The ESAP Smartwatch project is an open-source initiative by the Embedded Systems @ Purdue team to create a fully functional, customizable smartwatch with health tracking capabilities.

## Project Goals

Our mission is to provide an accessible, educational platform for learning about:
- Embedded systems design and development
- PCB design and manufacturing
- Bluetooth Low Energy (BLE) communication
- Mobile app development with React Native
- Health tracking algorithms and sensors
- Open-source hardware principles

## The Team

This project is developed and maintained by the **Embedded Systems @ Purdue** student organization, with contributions from engineering students passionate about creating innovative wearable technology.

**Project Lead**: William Zhang

## Key Features

### Hardware
- Custom-designed PCB using KiCad
- Heart rate monitoring sensor
- Accelerometer for movement tracking
- Bluetooth Low Energy connectivity
- Battery-optimized design

### Software
- Cross-platform mobile app (iOS & Android)
- Real-time data synchronization
- Workout logging and statistics
- Bluetooth device management
- User-friendly interface

## Open Source

All hardware designs, firmware, and mobile app code are available under the Apache 2.0 License, making this project freely accessible for:
- Educational purposes
- Personal modifications
- Research and development
- Community contributions

[View on GitHub](https://github.com/ESAP-Smartwatch/app)

## Technology Stack

- **Mobile**: React Native, Expo, JavaScript/TypeScript
- **Hardware**: Arduino, KiCad, Embedded C/C++
- **Communication**: Bluetooth Low Energy (BLE)
- **Design Tools**: KiCad for PCB, Figma for UI/UX
