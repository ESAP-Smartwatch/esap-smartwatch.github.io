### Installation Steps

1. **Clone the Repository**
   ```bash
   git clone https://github.com/ESAP-Smartwatch/app.git
   cd app
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run the App**
   ```bash
   npx expo start
   ```

## Technology Stack

### Mobile Development
- **React Native**: Cross-platform mobile framework
- **Expo**: Development and build tools
- **JavaScript/TypeScript**: Primary languages
- **React Navigation**: Screen navigation
- **Context API**: State management

### Hardware
- **Arduino**: Microcontroller platform
- **KiCad**: PCB design software
- **Bluetooth Low Energy**: Wireless communication
- **C/C++**: Embedded firmware

## API Reference

The app communicates with the smartwatch hardware via Bluetooth Low Energy (BLE):

- **Heart Rate Data**: Receives real-time heart rate measurements
- **Movement Data**: Tracks steps and activity levels
- **Device UUID**: Unique identifier for persistent pairing
- **Data Format**: UTF-8 encoded files
